# Francesco Dorati - Portfolio

A terminal-inspired personal portfolio showcasing experience in Robotics, Systems Engineering, and AI.

## 🚀 Tech Stack
- **Framework**: React 19 (Vite)
- **Routing**: React Router 7 (HashRouter for GitHub Pages)
- **Styling**: Pure CSS (Custom Theme)
- **Deployment**: GitHub Pages

## 📂 Project Structure
- `src/data/`: Centralized data store for projects, films, and books.
- `src/pages/`: Dynamic page components (About, Projects, Cinema, etc.).
- `src/components/`: Reusable UI components and layouts.

## 🛠 Usage
1.  `npm install`
2.  `npm run dev` (Local development)
3.  `npm run deploy` (Build and push to GitHub Pages)
